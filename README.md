# pc-spy
### 自制pc性能监视器
## 注意
### 这是一个模板，需要自己修改自己pc硬件名称条目，其中背景图是我用axure+ps做的。可自己根据需求改
### 不同pc硬件型号不同如我的电脑有8核故开启了8个核监视器 在aida64需根据自己硬件做出相应修改
### fps选项必须装微星 msi afterburner才有fps选项
